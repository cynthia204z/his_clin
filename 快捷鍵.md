##### window

`window` + `tab` 切換視窗

`window` + `g` 螢幕錄影

`window` + `shift` + `s` 螢幕截圖

`window` + `.` emoji menu





##### 常用

`shift` + `← / → / ↑ / ↓ ` 選取文字

`ctrl` + `← / → / ↑ / ↓ ` 以單字為單位移動游標

`ctrl` + `shift` + `→` 選取單字

`shift` + `home` 從游標位置選到行首

`shift` + `end` 從游標位置選到行尾

`home` 移到行首

`end` 移到行尾

`delete` 刪後面的字





##### 瀏覽器

`ctrl` + `tab` 切換標籤頁

`ctrl` + `w` 關閉當前標籤頁

`ctrl` + `t` 新增分頁
